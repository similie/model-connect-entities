Interface ISharedDataConnects<T>

ISharedDataConnects

Description

used by both the model class and connector class to give a common set of functionality between the two entities; Because the connector is often instantiated once. I have an extra param register to pass context to the connector class. Otherwise, it wouldn't know it's modelname or other identifies required to process the translation
