Class LiveConnectionAbstract

Name

LiveConnection

Description

The purpose of live connect is so that a single class can manage the start instance. Also, the child classes can safely choose to NOT implement certain functions such as init or teardown if they are not required. Additionally, for future functionality we can use this class to implement parent logic that all the child classes can extend without changing the structure of the child class. Finally, to create a new connector, you simply copy this class and populate the public functions with the desired logic
